### Installing in your browser

1. Open your web browser
2. Go to your browser's extensions page (e.g. `edge://extensions/`)
3. Enable "Developer mode"
4. Click on "Load unpacked"
5. Select the `build` directory from this project
6. The extension should now be installed and visible in your browser

## Privacy Policy

This extension does not collect, store, or transmit any personal data to external servers. All operations are performed locally on your device. For more details, see [privacy_policy.md](./privacy_policy.md).
